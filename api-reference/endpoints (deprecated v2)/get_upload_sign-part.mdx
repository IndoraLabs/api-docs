---
title: "GET /upload/sign-part"
---

**Tag:** Uploads

**Summary:** Get a signed URL for multipart upload chunk

**Description:** 

### Security
Bearer token (`Authorization: Bearer <JWT>`)

### Path
`/upload/sign-part`

### Method
`GET`

### Path & Query Parameters
| Name | In | Type | Required | Description |
|------|----|------|----------|-------------|
| `key` | query | string | True | S3 object key. |
| `uploadIdS3` | query | string | True | S3 UploadId from init. |
| `partNumber` | query | integer | True | Part number to upload (1-indexed). |

### Request Body


### 200 Response
| Field | Type | Description | Example |
|------|------|-------------|---------|
| `url` | string | Pre-signed URL. |  |

### Error Responses
- **400**: Bad request. Returns `Error` object.
- **401**: Unauthorized.
- **404**: Not found.
- **500**: Internal server error.
