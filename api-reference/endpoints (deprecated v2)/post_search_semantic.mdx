---
title: "POST /search/semantic"
---

**Tag:** Search

**Summary:** Hybrid search across uploaded content

**Description:** Perform hybrid semantic + keyword search across uploaded files. Supports case/file/date filters.

### Security
Bearer token (`Authorization: Bearer <JWT>`)

### Path
`/search/semantic`

### Method
`POST`

### Path & Query Parameters
_None_

### Request Body
| Field | Type | Description | Example |
|------|------|-------------|---------|
| `q` | string | Query string. Supports hybrid semantic + keyword search. |  |
| `caseId` | string (uuid) | Filter results to a specific case. |  |
| `fileId` | string (uuid) | Filter to a single file. |  |
| `limit` | integer | Number of results to return. |  |
| `dateFrom` | string (date-time) | Lower bound for file create/update timestamps. |  |
| `dateTo` | string (date-time) | Upper bound for file create/update timestamps. |  |

### 200 Response
| Field | Type | Description | Example |
|------|------|-------------|---------|
| `results` | array | Ordered search results. |  |

### Error Responses
- **400**: Bad request. Returns `Error` object.
- **401**: Unauthorized.
- **404**: Not found.
- **500**: Internal server error.
