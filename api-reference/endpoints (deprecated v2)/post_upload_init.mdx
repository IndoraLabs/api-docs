---
title: "POST /upload/init"
---

**Tag:** Uploads

**Summary:** Start an upload session

**Description:** Start an S3 upload session. Returns pre-signed URL for single PUT or UploadId for multipart.

### Security
Bearer token (`Authorization: Bearer <JWT>`)

### Path
`/upload/init`

### Method
`POST`

### Path & Query Parameters
_None_

### Request Body
| Field | Type | Description | Example |
|------|------|-------------|---------|
| `filename` | string | Client-provided file name (used for display only). |  |
| `contentType` | string | MIME type of the file (e.g., application/pdf). |  |
| `sizeBytes` | integer | Total file size in bytes. |  |

### 200 Response
| Field | Type | Description | Example |
|------|------|-------------|---------|
| `mode` | string | Upload mode chosen by server based on size. |  |
| `uploadId` | string | Server-side upload session id. |  |
| `key` | string | S3 object key to use for subsequent requests. |  |
| `putUrl` | string | Pre-signed URL for single part upload (present when mode=single). |  |
| `uploadIdS3` | string | S3 UploadId for multipart uploads. |  |

### Error Responses
- **400**: Bad request. Returns `Error` object.
- **401**: Unauthorized.
- **404**: Not found.
- **500**: Internal server error.
