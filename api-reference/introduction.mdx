## Welcome

The **Indora Labs API** is designed for both GovTech compliance and commercial use cases.  
It supports **240+ file types**, enabling:

- **Indexing**: Extract and embed content from text, audio, and video.  
- **Search**: Perform hybrid semantic + keyword queries.  
- **Redaction**: Auto-redact sensitive entities or retrieve redaction boxes for manual workflows.

## Authentication

All API endpoints require a **Bearer token**.  
Include it in your request headers:

```http
Authorization: Bearer <YOUR_API_KEY>
OpenAPI security configuration:

"security": [
  {
    "bearerAuth": []
  }
]
```
