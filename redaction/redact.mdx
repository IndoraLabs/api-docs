---
title: "Launch Redaction"
method: POST
path: /files/{fileId}/redact
---

Start an auto-redaction job on a file.  
Supports optional entity targeting (faces, plates, PII, etc.) and frame sampling rate.

## Authorizations
- **Authorization** (string, header, required)  
  Bearer authentication header: `Bearer <token>`

## Path Parameters
- **fileId** (string, required)  
  ID of the file to redact.

## Request Body
- **targets** (string, optional) – comma-separated list of entities to redact  
- **sampleFps** (integer, optional) – frame sampling rate (1–30)

## Response (202)
```json
{
  "ok": true,
  "jobId": "job-123",
  "status": "pending",
  "outputS3Uri": "s3://redacted/org/file.mp4"
}
